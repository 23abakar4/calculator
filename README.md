## About
This calculator is part of my small tools for local retail: it performs addition, subtraction, multiplication, division, modulus and power operations with a simple command-line menu. I built it to support my mother’s provisions shop in Aboabo, Kumasi.

---

## Features
- Console menu for operations: add, subtract, multiply, divide, modulus, power
- Simple text prompts for ease of use
- Designed for offline use (no internet required)
- Lightweight and easy to run on any machine with Python 3+
